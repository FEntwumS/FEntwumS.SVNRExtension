# FEntwumS.SVNRExtension
![image](https://github.com/FEntwumS/FEntwumS.SVNRExtension/blob/main/Icon.png?raw=true)

This plugin is an extension for OneWareStudio (OWS), designed to facilitate the conversion of Assembly code into a VHDL file format.

The target architecture is the SVNR (Simple Von Neumann Rechner), a straightforward 16-bit Von Neumann computer designed by the Technical University of Cologne (TH KÃ¶ln) specifically for educational purposes. 
This tool streamlines the process of translating low-level programs into hardware description language for simulation and implementation, enhancing the learning experience.
